# Morty
 Multi-module Jetpack Compose App with MVVM architecture.
 
 Lib Used:
 1. Ktor as HTTP API client,
 2. dagger hilt for DI,
 3. compose navigation,
 4. coil for image loading,
 5. material 3 design,
 6. splash screen API
